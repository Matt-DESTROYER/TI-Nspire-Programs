Markdown:

Markdown support is still a work-in-progress.

Bold:

Bold text can be created by surrounding text (on the same line) with asterisks.

Some *bold* text. → Some bold text.

Italic:

Italic text can be created by surrounding text (on the same line) with underscores.

Some _italic_ text. → Some italic text.

Code:

Simple code can be created by surrounding text (on the same line) with backticks.

Some `code`. → Some code

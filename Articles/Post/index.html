Markdown:

Markdown support is still a work-in-progress.

Headings:

Note: headings are not yet supported.

A heading can be created by starting a line with one or more hashtags (each additional hashtag reduces the size of the heading) and a space followed by text.

# Heading 1
## Heading 2
↓

Heading 1

Heading 2

Bold:

Bold text can be created by surrounding text (on the same line) with asterisks.

Some *bold* text. → Some bold text.

Italic:

Italic text can be created by surrounding text (on the same line) with underscores.

Some _italic_ text. → Some italic text.

Code:

Simple code can be created by surrounding text (on the same line) with backticks.

Some `code`. → Some code


Code blocks (which are NOT yet fully supported) can be created by surrounding text with three backticks.

```
some
code
```
↓

some
code

You can set the language used to enable syntax highlighting by writing the language next to the three backticks.

```lua
local string = "The quick brown fox jumped over the lazy dog.";
local PI = 3.14159;
```
↓

local string = "The quick brown fox jumped over the lazy dog.";
local PI = 3.14159;

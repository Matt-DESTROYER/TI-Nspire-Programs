TI-Nspire Programs - Luna        

Luna

One of the best tools to compile Lua programs into executables runnable by a TI-Nspire calculator is Luna.


How do I get Luna?

You can compile the latest version of Luna here or download Luna v0.2 (which works perfectly fine) below (simply unzip the downloaded file to use luna.exe).


I have Luna, what next? How do I use Luna?

Luna can be run from cmd (the command prompt) on a windows system likeso: luna.exe "lua program filename.lua" "output file filename.tns"

To learn about writing lua programs for Ti-Nspire calculators check out the tutorials.
